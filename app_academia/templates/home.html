Sobre esse projeto

O projeto academia é um site para cadastro e gerenciamento de treinos. Planejo usar ele para uso pessoal salvando meus treinos e exercícios da academia.

Esse é um projeto sem finalidade comercial nem fins lucrativos, continuamente em desenvolvimento. Através desse projeto busco aprimorar um pouco do meu conhecimento como desenvolvedor web e designer. As ferramentas utilizadas nesse projeto são principalmente HTML, CSS, JavaScript, Bootstrap, Python e Django. (Explicação) A terceira imagem é o meu shape dos sonhos.
